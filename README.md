#  Makamer — Personal Finance Tracker

A modern Android personal finance app built with Jetpack Compose. Track your income and expenses, manage your cards, and stay on top of your financial life, all with offline-first support.

---

##  Features

###  Authentication
- [ ] Sign up with Email & Password
- [ ] Login with Email & Password
- [ ] Sign in with Google

###  Home
- [ ] Display total Income and Expenses overview
- [ ] Show recent transactions list
- [ ] Quick add transaction button

###  Add Transaction
- [ ] Choose transaction type (Income / Expense)
- [ ] Enter amount
- [ ] Select category
- [ ] Pick a date
- [ ] Add a note
- [ ] Select the card used

###  Categories
- [ ] View all categories
- [ ] Add custom categories

###  Account
- [ ] View profile information
- [ ] View and manage physical & virtual cards

###  History
- [ ] View full transaction history with all details

###  Settings
- [ ] Toggle Light / Dark theme
- [ ] Change app language

---

##  Technologies Used

| Technology | Purpose |
|---|---|
| Kotlin | Primary language |
| Jetpack Compose | UI framework |
| Kotlinx Serialization | JSON serialization |
| Supabase | Backend, authentication & database |
| Room | Local database (offline-first) |
| Koin | Dependency injection |
| Navigation 3 | In-app navigation |
